# Promptic v0.1.3

## Highlights

- **Advanced Versioning System** â€“ Full support for customizable version delimiters (`_`, `.`, `-`), custom version patterns, prerelease postfixes (`-alpha`, `-beta`, `-rc.1`), and custom classifiers (e.g., language variants like `prompt_en_v1.md`, `prompt_ru_v1.md`). Configure via `VersioningConfig` with pydantic-settings integration.
- **Simplified API** â€“ New `render()` function for one-line prompt rendering: `render("prompts/task.md", vars={"key": "value"})`. No need to manually load networks or handle rendering separately.
- **Enhanced Variable Support** â€“ Variable substitution now works in `export_version()` function, allowing you to customize exported prompts with runtime values.
- **Comprehensive Documentation** â€“ Added guides for version classifiers, prereleases, and versioning configuration, plus complete API reference documentation.
- **Extensive Test Coverage** â€“ Added 1500+ lines of tests covering classifiers, custom patterns, delimiters, prereleases, and backward compatibility.

## Upgrade Notes

- **VersioningConfig is Optional** â€“ The new advanced versioning features are opt-in. Existing code continues to work without changes. Pass `VersioningConfig` explicitly when you need custom delimiters, patterns, or classifiers.
- **Default Behavior Unchanged** â€“ Underscore delimiter (`_v1`) remains the default. Projects using existing naming conventions don't need to migrate.
- **Classifier Ordering** â€“ When using classifiers with versions, classifiers must come BEFORE the version identifier: `prompt-en-v1.md` (not `prompt-v1-en.md`). Prerelease postfixes come AFTER: `prompt-en-v1-rc.1.md`.

## New Features

### Configurable Version Delimiters

```python
from promptic.versioning import VersioningConfig, load_prompt

# Use hyphen delimiter
config = VersioningConfig(delimiter="-")
prompt = load_prompt("prompts", version="latest", config=config)
# Matches: prompt-v1.md, prompt-v2.md

# Use dot delimiter
config = VersioningConfig(delimiter=".")
prompt = load_prompt("prompts", version="latest", config=config)
# Matches: prompt.v1.md, prompt.v2.md

# Support multiple delimiters
config = VersioningConfig(delimiters=["_", "-"])
# Matches both: prompt_v1.md and prompt-v1.md
```

### Custom Version Patterns

```python
# Custom pattern: "rev1", "rev2", etc.
config = VersioningConfig(
    pattern=r"rev(\d+)",
    version_parser=lambda m: f"v{m.group(1)}.0.0"
)
prompt = load_prompt("prompts", version="latest", config=config)
# Matches: prompt_rev1.md, prompt_rev2.md
```

### Language Classifiers

```python
# Load Russian version specifically
config = VersioningConfig(classifiers=["lang"])
prompt = load_prompt(
    "prompts",
    version="latest",
    classifier_values={"lang": "ru"},
    config=config
)
# Matches: prompt_ru_v1.md, prompt_ru_v2.md
# Returns latest Russian version even if newer English versions exist
```

### Prerelease Support

```python
# Load latest stable (excludes prereleases)
prompt = load_prompt("prompts", version="latest")
# Matches: prompt_v1.0.0.md (not prompt_v1.0.0-alpha.md)

# Load latest including prereleases
prompt = load_prompt("prompts", version="latest", include_prerelease=True)
# Matches: prompt_v1.0.0-rc.1.md if it's the newest
```

### Simplified render() Function

```python
from promptic import render

# Simple rendering
output = render("prompts/task.md")

# With variables
output = render(
    "prompts/task.md",
    vars={"user_name": "Alice", "task_type": "analysis"}
)

# Different format and mode
yaml_output = render(
    "prompts/task.md",
    target_format="yaml",
    render_mode="file_first"
)
```
