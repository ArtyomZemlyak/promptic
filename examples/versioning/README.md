# Prompt Versioning Examples

This directory contains examples of using the prompt versioning system.

## Contents

- `basic/`: Simple example with versioned prompts in a flat directory.
- `hierarchical/`: Example with nested prompt references and hierarchical versioning.
- `export_script.py`: Python script demonstrating how to load and export versions.

## Running the Examples

You can run the export script to see versioning in action:

```bash
python examples/versioning/export_script.py
```
