# Формат памяти

## Структура памяти для агентов

Память используется для хранения контекста между сессиями и для поиска связанной информации.

### Формат записи

```json
{
  "id": "unique_id",
  "type": "note|media|git_operation",
  "content": "основное содержимое",
  "metadata": {
    "created_at": "YYYY-MM-DD HH:MM:SS",
    "updated_at": "YYYY-MM-DD HH:MM:SS",
    "tags": ["tag1", "tag2"],
    "related": ["id1", "id2"]
  },
  "vector": [0.1, 0.2, ...]
}
```

### Типы записей

1. **note**: Заметки из knowledge base
2. **media**: Обработанные медиа-файлы
3. **git_operation**: История Git операций

### Использование

- Поиск похожих заметок
- Поиск связанных медиа-файлов
- История операций
- Контекст для новых запросов
