# Обработка видео

## Задача

Обработать видео-файл: извлечь транскрипцию, ключевые кадры и сохранить результат в структурированном формате.

## Процесс обработки

1. **Подготовка видео**
   - Проверить доступность файла
   - Определить формат, длительность, разрешение
   - Проверить качество видео

2. **Извлечение транскрипции**
   - Использовать инструменты для извлечения аудио
   - Транскрибировать аудио-дорожку
   - Сохранить с временными метками

3. **Извлечение ключевых кадров**
   - Определить важные моменты
   - Извлечь кадры в ключевые точки
   - Сохранить описания кадров

4. **Структурирование результата**
   - Создать Markdown файл в директории media/ с именем исходного файла
   - Сохранить транскрипцию
   - Добавить описания ключевых кадров
   - Организовать по временным меткам

5. **Добавление метаданных**
   - Дата обработки
   - Длительность видео
   - Разрешение
   - Формат файла
   - Инструменты обработки

## Формат результата

Результат сохраняется в файл в директории media/:

```markdown
# Обработка видео

## Транскрипция

[00:00] Начало видео...

[01:30] Ключевой момент...

## Ключевые кадры

- [00:15] Описание кадра 1
- [01:30] Описание кадра 2

## Метаданные
- Дата обработки: YYYY-MM-DD
- Длительность: MM:SS
- Разрешение: WIDTHxHEIGHT
- Формат: VIDEO_FORMAT
```
