# Транскрибация аудио

## Задача

Транскрибировать аудио-файл и сохранить результат в структурированном Markdown формате.

## Процесс транскрибации

1. **Подготовка аудио**
   - Проверить доступность файла
   - Определить формат и длительность
   - Проверить качество записи

2. **Выполнение транскрибации**
   - Использовать инструменты транскрибации
   - Распознать речь из аудио
   - Сохранить структуру (разделы, спикеры если есть)

3. **Структурирование результата**
   - Создать Markdown файл в директории media/ с именем исходного файла
   - Сохранить транскрипцию с временными метками
   - Разделить по спикерам если несколько
   - Добавить заголовки для разделов

4. **Добавление метаданных**
   - Дата обработки
   - Длительность аудио
   - Формат файла
   - Инструмент транскрибации

## Формат результата

Результат сохраняется в файл в директории media/:

```markdown
# Транскрипция аудио

## Основной текст

[00:00] Начало записи...

[00:30] Продолжение...

## Метаданные
- Дата обработки: YYYY-MM-DD
- Длительность: MM:SS
- Формат: AUDIO_FORMAT
- Инструмент: TRANSCRIPTION_TOOL
```
