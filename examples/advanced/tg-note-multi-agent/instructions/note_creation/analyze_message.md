# Анализ сообщения пользователя

## Задача

Проанализировать входящее сообщение от пользователя и определить:
- Тип запроса (создание заметки, обновление, поиск)
- Ключевые темы и концепции
- Необходимые действия
- Связанные медиа-файлы если есть

## Процесс анализа

1. **Определение типа запроса**
   - Создание новой заметки
   - Обновление существующей
   - Поиск информации
   - Обработка медиа

2. **Извлечение ключевой информации**
   - Основная тема
   - Важные детали
   - Контекст и предыстория
   - Связанные концепции

3. **Определение структуры**
   - Какие разделы нужны
   - Какие медиа-файлы обработать
   - Какие ссылки добавить

4. **Проверка существующих заметок**
   - Есть ли похожие заметки
   - Нужно ли обновить существующую
   - Какие связи установить
