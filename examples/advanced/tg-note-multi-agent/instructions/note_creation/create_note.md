# Создание заметки

## Задача

Создать структурированную заметку в формате Markdown на основе
проанализированного сообщения пользователя.

## Процесс создания

1. **Структурирование контента**
   - Создать заголовок (H1)
   - Разбить на логические разделы
   - Добавить подзаголовки где нужно
   - Организовать информацию иерархически

2. **Форматирование**
   - Применить правила Markdown форматирования
   - Добавить списки для перечислений
   - Использовать выделение для важных моментов
   - Добавить код-блоки если нужно

3. **Обработка медиа**
   - Если есть медиа-файлы, добавить ссылки на них
   - Использовать формат: `![описание](media/{filename}.md)`
   - Для OCR результатов читать `media/{filename}.md`

4. **Добавление метаданных**
   - Дата создания
   - Теги/категории
   - Связанные заметки
   - Автор (если известен)

5. **Проверка качества**
   - Проверить структуру
   - Убедиться в читаемости
   - Проверить ссылки
   - Убедиться в полноте информации
