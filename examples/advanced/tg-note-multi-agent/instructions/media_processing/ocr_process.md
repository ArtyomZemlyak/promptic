# OCR Процесс

## Задача

Выполнить OCR (Optical Character Recognition) для изображения и сохранить
результат в структурированном Markdown формате.

## Процесс OCR

1. **Подготовка изображения**
   - Проверить доступность файла
   - Определить формат и размер
   - Проверить качество изображения

2. **Выполнение OCR**
   - Использовать инструменты OCR (qwen code cli или другие)
   - Распознать текст из изображения
   - Сохранить структуру (заголовки, списки, таблицы)

3. **Структурирование результата**
   - Создать Markdown файл: `media/{filename}.md`
   - Сохранить заголовки если распознаны
   - Сохранить основной текст
   - Сохранить списки и таблицы в Markdown формате

4. **Добавление метаданных**
   - Дата обработки
   - Размер изображения
   - Качество распознавания
   - Инструмент OCR

## Формат результата

Результат сохраняется в `media/{filename}.md` в следующем формате:

```markdown
# Заголовок (если распознан)

Основной текст распознанный из изображения.

## Метаданные
- Дата обработки: YYYY-MM-DD
- Размер изображения: WIDTHxHEIGHT
- Инструмент: OCR_TOOL_NAME
```
