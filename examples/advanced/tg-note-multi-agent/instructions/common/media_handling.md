# Обработка медиа-файлов

## Общие правила работы с медиа

### Структура файлов

Медиа-файлы обрабатываются и сохраняются в директории `media/`:
- Изображения → `media/{filename}.md` (OCR результат)
- Аудио → `media/{filename}.md` (транскрипция)
- Видео → `media/{filename}.md` (транскрипция + ключевые кадры)

### Доступ к результатам обработки

Если нужно получить OCR или транскрипцию конкретного медиа:
1. Замени расширение файла на `.md`
2. Прочитай файл: `media/{filename}.md`
3. Там будет структурированный результат обработки

### Формат результатов

Все результаты сохраняются в Markdown формате:
- Заголовок из медиа (если есть)
- Основной текст/транскрипция
- Метаданные (дата, размер, тип)
- Списки и таблицы если распознаны

### Пример структуры OCR файла

```markdown
# Заголовок из изображения

Основной текст распознанный из изображения.

## Метаданные
- Дата обработки: 2025-01-28
- Тип: изображение
- Размер: 1920x1080
```
