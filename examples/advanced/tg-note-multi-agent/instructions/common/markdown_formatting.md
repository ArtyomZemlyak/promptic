# Markdown Форматирование

## Общие правила форматирования заметок

### Структура документа

1. **Заголовки**: Используй иерархию заголовков (H1-H6) для структурирования
2. **Списки**: Используй маркированные или нумерованные списки для перечислений
3. **Код**: Обрамляй код в тройные backticks с указанием языка
4. **Ссылки**: Используй Markdown синтаксис для ссылок `[текст](url)`

### Стиль написания

- Пиши кратко и по делу
- Используй активный залог
- Добавляй примеры где уместно
- Структурируй информацию логически

### Специальные элементы

- **Таблицы**: Используй Markdown таблицы для структурированных данных
- **Цитаты**: Используй `>` для цитат
- **Выделение**: Используй **жирный** и *курсив* для акцентов
