# Proofread for Errors

Conduct final proofreading to catch and correct all errors.

## Proofreading Focus

1. **Spelling**: Check all words
2. **Grammar**: Verify sentence structure
3. **Punctuation**: Ensure correct usage
4. **Consistency**: Check terminology and style
5. **Formatting**: Verify formatting is correct

## Common Errors to Check

- Spelling mistakes and typos
- Grammar errors
- Punctuation mistakes
- Inconsistent capitalization
- Formatting inconsistencies
- Broken links
- Missing or incorrect citations

## Proofreading Techniques

- Read slowly and carefully
- Read aloud to catch errors
- Use spell-check tools
- Check one type of error at a time
- Have fresh eyes (take breaks)
- Read backwards for spelling

## Final Checks

- All names spelled correctly
- Numbers and dates accurate
- Links work properly
- Formatting is consistent
- No placeholder text remains
- Citations are complete
