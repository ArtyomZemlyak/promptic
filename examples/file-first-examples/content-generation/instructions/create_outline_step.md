# Create Content Outline

Develop a structured outline that guides content creation.

## Outline Purpose

- Organize information logically
- Ensure all key points are covered
- Maintain focus and flow
- Guide writing process
- Meet length requirements

## Outline Structure

1. **Introduction**: Hook, context, preview
2. **Main sections**: 3-5 key sections
3. **Subsections**: Supporting points within sections
4. **Conclusion**: Summary, call-to-action
5. **Supporting elements**: Examples, visuals, quotes

## Outline Development

- Start with key messages
- Group related ideas
- Order logically (chronological, problem-solution, etc.)
- Balance section lengths
- Include transitions

## Outline Validation

- Covers all key messages
- Logical flow and progression
- Appropriate depth for each section
- Meets format requirements
- Aligns with audience needs
