# Verify Facts and Statistics

Ensure all factual claims are accurate and verifiable.

## Fact Verification Process

1. **Cross-reference**: Check multiple sources
2. **Check dates**: Verify statistics are current
3. **Verify numbers**: Confirm calculations and data
4. **Source check**: Ensure sources are credible
5. **Context check**: Understand full context

## Common Fact-Checking Areas

- Statistics and percentages
- Dates and timelines
- Names and titles
- Quotes and attributions
- Technical specifications
- Historical events

## Verification Methods

- Cross-reference with original sources
- Check official databases
- Consult subject matter experts
- Use fact-checking tools
- Review recent updates

## Red Flags

- Numbers that seem too good to be true
- Outdated statistics
- Unattributed claims
- Conflicting information across sources
- Single-source claims

## Documentation

- Record verification status
- Note source for each fact
- Document any uncertainties
- Flag items needing updates
- Keep verification trail
