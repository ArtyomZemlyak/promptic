# Add Visual Elements

Incorporate visual elements that enhance content effectiveness.

## Visual Types

1. **Images**: Photos, illustrations, graphics
2. **Charts and graphs**: Data visualizations
3. **Infographics**: Information graphics
4. **Videos**: Embedded or linked videos
5. **Interactive elements**: Quizzes, calculators

## Visual Selection

- Relevant to content
- High quality and professional
- Supports key messages
- Enhances understanding
- Appropriate for audience

## Visual Placement

- Break up text blocks
- Support nearby content
- Guide reader attention
- Maintain flow
- Balance with text

## Visual Best Practices

- Use original or licensed images
- Optimize file sizes
- Include alt text for accessibility
- Ensure mobile responsiveness
- Maintain brand consistency

## Visual Integration

- Reference visuals in text
- Use captions when helpful
- Ensure visual-text alignment
- Create visual hierarchy
- Test on different devices
