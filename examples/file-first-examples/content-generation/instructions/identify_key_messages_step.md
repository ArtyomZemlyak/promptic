# Identify Key Messages

Extract and prioritize the core messages that must be communicated.

## Message Identification

1. **Primary message**: The main point to convey
2. **Supporting messages**: Additional important points
3. **Call-to-action**: Desired reader response
4. **Brand messages**: Brand positioning elements
5. **Value propositions**: Benefits to highlight

## Message Prioritization

Rank messages by:
- Importance to objectives
- Relevance to audience
- Uniqueness or differentiation
- Actionability
- Memorability

## Message Development

For each key message:
- Define the core idea clearly
- Identify supporting evidence
- Consider audience perspective
- Ensure alignment with brand
- Make it memorable and actionable

## Message Integration

- Weave messages throughout content naturally
- Avoid being overly promotional
- Support with examples and stories
- Reinforce through repetition (subtle)
- End with clear call-to-action
