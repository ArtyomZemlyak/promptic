# Generate Insights and Recommendations

Synthesize analysis findings into actionable business insights.

## Insight Generation Process

1. **Summarize key findings**: Main patterns and relationships
2. **Prioritize by impact**: Focus on most significant insights
3. **Provide context**: Explain why findings matter
4. **Create recommendations**: Suggest specific actions
5. **Quantify value**: Estimate potential impact

## Insight Structure

Each insight should include:
- **Finding**: What was discovered
- **Evidence**: Data supporting the finding
- **Significance**: Why it matters
- **Recommendation**: What to do about it
- **Expected impact**: Potential outcome

## Best Practices

- Focus on actionable insights
- Use clear, non-technical language
- Support with visualizations
- Prioritize by business value
- Include confidence levels
