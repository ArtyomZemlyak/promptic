# Synthesize Findings

Combine individual analysis results into coherent narrative.

## Synthesis Process

1. **Group related findings**: Cluster by theme or topic
2. **Identify patterns**: Look for overarching themes
3. **Resolve contradictions**: Address conflicting findings
4. **Build narrative**: Create logical flow
5. **Highlight key takeaways**: Emphasize most important points

## Synthesis Structure

- **Executive summary**: High-level overview
- **Detailed findings**: Specific results by category
- **Patterns and themes**: Cross-cutting insights
- **Limitations**: Known constraints and caveats
- **Next steps**: Recommended follow-up analysis

## Quality Checks

- All major findings included
- Logical flow and organization
- Clear connections between findings
- Balanced presentation
- Actionable conclusions
