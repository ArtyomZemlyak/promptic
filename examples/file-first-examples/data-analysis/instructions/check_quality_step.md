# Check Data Quality

Assess data quality across multiple dimensions.

## Quality Dimensions

1. **Completeness**: Percentage of non-null values
2. **Accuracy**: Correctness of values
3. **Consistency**: Uniformity across records
4. **Timeliness**: Data freshness and relevance
5. **Validity**: Conformance to business rules

## Quality Checks

- Missing value analysis
- Duplicate detection
- Outlier identification
- Format consistency checks
- Cross-field validation

## Quality Metrics

Calculate and report:
- Completeness percentage per field
- Duplicate record count
- Outlier percentage
- Invalid value count
- Overall quality score

## Quality Thresholds

- **High quality**: >95% completeness, <1% duplicates
- **Medium quality**: 80-95% completeness, 1-5% duplicates
- **Low quality**: <80% completeness, >5% duplicates
