# Trend Analysis

Identify and analyze trends over time or across sequences.

## Trend Types

1. **Linear trends**: Steady increase or decrease
2. **Exponential trends**: Accelerating growth or decay
3. **Seasonal trends**: Recurring patterns (daily, weekly, yearly)
4. **Cyclical trends**: Long-term cycles
5. **Structural breaks**: Sudden changes in pattern

## Analysis Techniques

- Time series decomposition
- Moving averages
- Regression analysis
- Change point detection
- Seasonal adjustment

## Trend Metrics

- Direction (increasing, decreasing, stable)
- Magnitude (rate of change)
- Consistency (steady vs volatile)
- Duration (short-term vs long-term)
- Significance (statistical validation)

## Documentation

Record:
- Trend type identified
- Statistical measures
- Visual evidence
- Domain interpretation
- Forecast implications
