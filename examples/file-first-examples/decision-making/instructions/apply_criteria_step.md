# Apply Decision Criteria

Systematically evaluate options against defined decision criteria.

## Criteria Selection

Choose criteria that:
- Align with objectives
- Are measurable or assessable
- Cover key considerations
- Are relevant to all options
- Are prioritized appropriately

## Common Criteria Types

- **Strategic alignment**: Fit with goals
- **Financial impact**: Cost and value
- **Risk level**: Likelihood and impact
- **Feasibility**: Ease of implementation
- **Timeline**: Speed to results
- **Quality**: Expected outcomes
- **Stakeholder support**: Acceptance level

## Criteria Weighting

Assign weights based on:
- Importance to objectives
- Strategic priorities
- Stakeholder values
- Context and constraints
- Relative significance

## Criteria Application

- Apply consistently to all options
- Use clear scoring method
- Document rationale for scores
- Consider both quantitative and qualitative
- Review for bias or inconsistency

## Criteria Documentation

- List all criteria used
- Document weights assigned
- Record scores for each option
- Explain scoring rationale
- Note any adjustments made
