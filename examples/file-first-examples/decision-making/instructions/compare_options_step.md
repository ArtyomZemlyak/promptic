# Compare Options Systematically

Compare all options side-by-side using consistent criteria.

## Comparison Framework

1. **Select criteria**: Choose relevant evaluation dimensions
2. **Weight criteria**: Assign importance to each
3. **Score options**: Rate each option on each criterion
4. **Calculate totals**: Weighted scores
5. **Analyze differences**: Identify key distinctions

## Comparison Process

- Use same criteria for all options
- Apply consistent scoring method
- Consider both quantitative and qualitative factors
- Document all assessments
- Review for fairness and completeness

## Comparison Visualization

- Create comparison matrix
- Use side-by-side tables
- Highlight key differences
- Show trade-offs clearly
- Visualize scores and rankings

## Key Comparisons

- Benefits vs costs
- Risks vs rewards
- Feasibility vs ambition
- Short-term vs long-term
- Certainty vs potential

## Comparison Documentation

- Document all scores and weights
- Explain key differences
- Note trade-offs
- Highlight strengths and weaknesses
- Record any assumptions
