# Collect Data and Evidence

Systematically gather quantitative and qualitative data.

## Data Types

1. **Quantitative**: Numbers, metrics, statistics
2. **Qualitative**: Observations, descriptions, narratives
3. **Primary**: Original data collected directly
4. **Secondary**: Data from existing sources
5. **Benchmark**: Comparative data from others

## Data Collection Methods

- Surveys and questionnaires
- Interviews and focus groups
- Observations and measurements
- Document review
- Database queries
- Market research

## Data Quality Checks

- Verify data accuracy
- Check for completeness
- Assess relevance
- Validate sources
- Identify biases or limitations

## Data Organization

- Structure data logically
- Create summaries and visualizations
- Document sources and methods
- Note any limitations
- Prepare for analysis

## Data Analysis Preparation

- Clean and prepare data
- Identify patterns and trends
- Calculate key metrics
- Create visualizations
- Summarize key findings
