# Evaluate Each Option

Systematically assess each option against relevant criteria.

## Evaluation Framework

1. **Feasibility**: Can it be done?
2. **Benefits**: What value does it provide?
3. **Risks**: What could go wrong?
4. **Costs**: What resources are required?
5. **Alignment**: Does it meet objectives?

## Evaluation Process

- Apply consistent criteria to all options
- Gather evidence for each criterion
- Assess objectively and fairly
- Document findings clearly
- Compare across options

## Evaluation Depth

- **Quick scan**: Initial feasibility check
- **Detailed analysis**: Comprehensive assessment
- **Deep dive**: In-depth examination for finalists
- **Validation**: Expert review and confirmation

## Evaluation Documentation

For each option:
- Feasibility assessment
- Benefits analysis
- Risk assessment
- Cost estimate
- Alignment with objectives
- Overall evaluation summary

## Quality Checks

- Evaluation is thorough and fair
- Evidence supports assessments
- Criteria applied consistently
- Biases identified and mitigated
- Gaps in information noted
