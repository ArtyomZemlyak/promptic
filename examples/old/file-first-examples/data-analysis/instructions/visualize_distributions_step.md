# Visualize Distributions

Create visualizations to understand data distributions and patterns.

## Visualization Types

### Numeric Variables
- Histograms: Show frequency distribution
- Box plots: Display quartiles and outliers
- Density plots: Show probability distribution
- Q-Q plots: Check normality

### Categorical Variables
- Bar charts: Show frequency counts
- Pie charts: Show proportions (use sparingly)
- Treemaps: Show hierarchical categories

### Relationships
- Scatter plots: Two numeric variables
- Heatmaps: Correlation matrices
- Pair plots: Multiple variable relationships

## Best Practices

- Choose appropriate chart types
- Use clear labels and titles
- Apply consistent color schemes
- Include summary statistics
- Highlight important patterns
