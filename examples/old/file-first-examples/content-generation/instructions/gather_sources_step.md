# Gather Reliable Sources

Identify and collect information from credible, authoritative sources.

## Source Types

1. **Academic sources**: Research papers, journals, studies
2. **Industry reports**: Market research, industry analysis
3. **Expert interviews**: Direct quotes from authorities
4. **Government data**: Official statistics and reports
5. **Reputable publications**: Established news and media

## Source Evaluation

Assess each source for:
- **Authority**: Author credentials and expertise
- **Accuracy**: Factual correctness
- **Objectivity**: Balanced perspective
- **Currency**: Publication date and relevance
- **Coverage**: Depth and completeness

## Source Collection

- Document source URLs and citations
- Save quotes and key statistics
- Note publication dates
- Record author information
- Organize by topic or theme

## Source Diversity

- Include multiple perspectives
- Balance different viewpoints
- Use both primary and secondary sources
- Mix quantitative and qualitative data
- Consider international sources
