# Structure Content Flow

Organize content in a logical, engaging flow that guides readers.

## Flow Patterns

1. **Problem-solution**: Identify problem, present solution
2. **Chronological**: Time-based progression
3. **Hierarchical**: General to specific
4. **Comparative**: Compare options or approaches
5. **Narrative**: Story-driven structure

## Structure Elements

- **Opening hook**: Engage immediately
- **Clear progression**: Logical step-by-step flow
- **Smooth transitions**: Connect sections naturally
- **Balanced sections**: Similar depth and length
- **Strong conclusion**: Memorable ending

## Flow Optimization

- Start with most important information
- Use headings to guide navigation
- Include visual breaks (lists, quotes, images)
- Vary sentence and paragraph length
- Build momentum toward conclusion

## Reader Experience

- Easy to scan and navigate
- Clear value at each section
- Natural reading rhythm
- Engaging and interesting
- Actionable takeaways
