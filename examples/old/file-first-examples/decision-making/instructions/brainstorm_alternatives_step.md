# Brainstorm Alternatives

Generate creative and diverse alternatives without initial judgment.

## Brainstorming Process

1. **Set ground rules**: Encourage all ideas, defer judgment
2. **Generate freely**: Produce many ideas quickly
3. **Build on ideas**: Combine and extend suggestions
4. **Record everything**: Capture all ideas
5. **Review and refine**: Organize and improve ideas

## Brainstorming Techniques

- **Free association**: Start with one idea, branch out
- **Role playing**: Consider different perspectives
- **What-if scenarios**: Explore hypothetical situations
- **Reverse thinking**: Consider opposite approaches
- **Analogies**: Learn from similar situations

## Brainstorming Best Practices

- Create safe, judgment-free environment
- Encourage quantity over quality initially
- Build on others' ideas
- Include diverse perspectives
- Set time limits to maintain energy

## Idea Capture

- Document all ideas
- Use clear, descriptive names
- Note key characteristics
- Capture any initial thoughts on feasibility
- Organize by theme or category

## Post-Brainstorming

- Review all ideas generated
- Identify most promising
- Combine similar ideas
- Refine and develop further
- Prepare for evaluation
